Sensitivity analysis on the STICS model using the sensitive_stics() and save_sensi() functions from the sticRs package (https://github.com/VEZY/sticRs).
Parameters used:
P_rapforme
Output variables on which the effect of the parameters where tested:
hauteur, lai(n), masec(n)
Please use the sticRs::load_sensi() function to load the sensitivity analysis back to R.
